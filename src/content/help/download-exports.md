# Where do I download my Hostaway/Airbnb/QuickBooks exports?

Each platform stores its CSV exports in a slightly different location. Here is the quickest path on each one.

## Hostaway

Sign in to Hostaway and open Reports from the left navigation. Choose "Consolidated financial report", set the date range you want to include, and click "Export CSV" in the top-right corner. The download will include reservations, payouts, and channel fees in a single file that StrataReport AI can ingest directly.

## Airbnb

From the Airbnb host dashboard, open Menu, then Transaction history. Switch to the "Completed payouts" tab, choose the relevant year, and click "Download CSV". For tax season, also download the "Earnings summary" CSV from the same page.

## QuickBooks Online

In QuickBooks Online, open Reports and search for "Profit and Loss by Class" or "Transaction List by Date". Run the report for the period you want, then use the "Export" button at the top of the report to download a CSV (not an Excel file). Make sure each property is tagged with a class so the report engine can split the figures by property.

If your platform is not listed, any CSV with the required columns will work.
